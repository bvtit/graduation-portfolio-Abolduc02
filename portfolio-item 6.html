Project Description

A local car retail company, Imperial Cars, had an aging phone and computer system that desperately needed to be replaced to reduce time wasted due to technical issues which would help lead to better work production. Two other co-workers and I were put to the task of preparing and setting up each new PC and phone that was delivered for Imperial Cars. There were around 125 new computers that needed to be prepared and somewhere around 250 new phones. For the computers, we had to turn them on, complete basic setup and add to the domain, and install needed software. For the phones, we had to plug them into the network so they could reach the provisioning server to download any configurations that they needed. We prepped everything at our office to make sure that everything was all set before we deployed any of the devices into the work environment.

Project Details

  • Polycom VVX 410 Phones
  • VOIP system configuration and security
  • Computer software installation
  • Basic computer setup/updates
